# ☕ Cafe Bot — Quick Start

## Run the bot
1. Open VS Code
2. File → Open Folder → select this "Cafe Bot" folder
3. Press Ctrl + ` to open the terminal
4. Type: pip install -r requirements.txt  (first time only)
5. Type: python bot.py

You should see: ✅ CafeBot is online and ready!

## Commands
!balance        — Check your Gold & XP
!duel @user     — Challenge someone to a duel
!accept         — Accept a duel challenge
!claim_win      — Collect your Gold after winning
!leaderboard gold  — Gold leaderboard
!leaderboard xp    — XP leaderboard
!lfg Valorant Looking for 2 players  — Post LFG listing

## Admin commands
!give @user 100   — Give Gold
!take @user 50    — Remove Gold
!updateroles      — Refresh all roles
